🍃 MongoDB + Vercel - نظام قاعدة البيانات الاحترافي

✅ نظام احترافي مع MongoDB و Vercel Functions!

📋 الخطوات المطلوبة:

1️⃣ إعداد MongoDB:
   - اذهب إلى mongodb.com/atlas
   - أنشئ حساب مجاني
   - أنشئ cluster جديد
   - احصل على Connection String

2️⃣ إعداد Vercel:
   - اذهب إلى vercel.com
   - سجل حساب جديد (مجاني)
   - اربط حساب GitHub
   - ارفع المشروع

3️⃣ إعداد Environment Variables:
   - في Vercel Dashboard
   - اذهب إلى Settings > Environment Variables
   - أضف MONGODB_URI مع Connection String

4️⃣ تحديث رابط API:
   - غيّر رابط API في mongodb-sync.js
   - استخدم رابط Vercel الخاص بك
